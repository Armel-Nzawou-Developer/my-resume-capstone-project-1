CSS Grid & Flexbox Cheat Sheet | by Armel Nzawou

CSS Grid - Container Properties

  • display: grid | inline-grid
  • grid-template-columns / grid-template-rows
    • Example: grid-template-columns: repeat(3, 1fr);
  • grid-template-areas
  • grid-template
  • grid-column-gap / grid-row-gap / gap
  • justify-items → start | end | center | stretch
  • align-items → start | end | center | stretch
  • place-items → shorthand for align-items + justify-items
  • justify-content → start | end | center | space-between | space-around | space-evenly
  • align-content → same values as justify-content
  • place-content → shorthand for align-content + justify-content
  • grid-auto-columns / grid-auto-rows
  • grid-auto-flow → row | column | dense

CSS Grid - Items properties

  • grid-column-start / grid-column-end
  • grid-row-start / grid-row-end
  • grid-column / grid-row → shorthand
  • grid-area
  • justify-self → start | end | center | stretch
  • align-self → same values as above
  • place-self → shorthand for align-self + justify-self

Flexbox - Container properties

  • display: flex | inline-flex
  • flex-direction → row | row-reverse | column | column-reverse
  • flex-wrap → nowrap | wrap | wrap-reverse
  • flex-flow → shorthand for flex-direction + flex-wrap
  • justify-content → flex-start | flex-end | center | space-between | space-around | space-evenly
  • align-items → flex-start | flex-end | center | baseline | stretch
  • align-content → flex-start | flex-end | center | space-between | space-around | stretch
  • gap → spacing between flex items

Flexbox - Items properties

  • order
  • flex-grow
  • flex → shorthand for grow, shrink, basis
  • align-self → auto | flex-start | flex-end | center | baseline | stretch
